# Search Hacker News

An app that searches the Hacker News

## Running Guide

### Step-1: Clone the project or Download the zip

To clone, run:

https method
```
git clone https://github.com/iamdipanshusingh/Hacker-News-Assignment-Flutter.git
```

OR

ssh
```
git clone git@github.com:iamdipanshusingh/Hacker-News-Assignment-Flutter.git
```

### Step-2: Get the required packages
```
flutter pub get
```

### Step-3: Run the project
```
flutter run
```
